# Student Management System (Flask & SQLite)

This project is a Student Management System built using Flask and SQLite.
It is designed to manage student records efficiently with authentication and CRUD operations.

# Features

User Login & Registration

Add Student Details

Edit Student Information

Delete Student Records

Search Students

Flash Messages for alerts & feedback

Clean and simple UI

# Technologies Used

Python (Flask)

SQLite

HTML5

CSS3

JavaScript

Visual Studio Code
